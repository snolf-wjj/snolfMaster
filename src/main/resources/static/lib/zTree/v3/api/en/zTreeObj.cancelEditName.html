Function(newName)zTreeObj.cancelEditName

Overview[ depends on jquery.ztree.exedit js ]

Cancel the edit text status. Can restore the original text, and can also force assigned to a new text.

Please use zTree object to executing the method.

Function Parameter Descriptions

newNameString

Re given a new text

If this parameter is omitted, then restore the original text.

Return none

no return value

Examples of function

1. Cancel edit text, and restore the original text.

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.cancelEditName();

2. Cancel edit text , and set the new text.

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.cancelEditName("test_new_name");
