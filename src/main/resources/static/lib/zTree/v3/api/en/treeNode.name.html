StringtreeNode.text

Overview[ depends on jquery.ztree.core js ]

The node's text

1. If you want to change 'text' attribute, please modify the 'setting.data.roleKey.text' attribute.

Default: undenfined

String Format

String object. The HTML special characters are escaped

Examples of treeNode

1. Set node's text to 'test1', 'test2', 'test3'

var children = [
	{ "id":1, "text":"test1"},
	{ "id":2, "text":"test2"},
	{ "id":3, "text":"test3"}
]
